Power Link

B2B message exchange management platform

Due to a swift rise of B2Bi e-commerce, a demand on message exchange and integration over internet among businesses, customers and suppliers is increasing from time to time. With its outstanding R&D capability, AboveE unveils an innovative Power Link message exchange management platform, by supporting RosettaNet and AS2 (Applicability Statement 2) and other global industry transaction standards along with its rapid adoption service, Power Link helps business construct a brand-new business transaction model within a short time, and significantly enhance the competitiveness of business in global industry transaction network.

 

With Power Link message exchange management platform, business will be able to set up and execute the various information exchanged with partners in an easy way, where the transparent query tracking capability gives you a complete process execution status, contents and status of exchanged message, allows you to track error message during the exchange process, etc. Moreover, it provides simple web management interface, where the embedded swift Level-1 and Level-2 testing mechanism could largely reduce the class-taking of system administrator which normally takes few weeks to learn as well as save the system maintenance cost. No matter the system administrators or internal users of B2B will be able to proceed quick query and conduct detailed tracking via browser to swiftly react to the issues between partners when under construction or when it goes live.

Power Link Infrastructure
 

Simple System Integration Method

In addition to data exchange between partners, Power Link even provides 3 methods to enable direct integration between partners’ data and enterprises’ back-end system.

 
1. Adapter manager embedded :Being responsible for all customized adapter connection with back-end system to customize as per real needs of the organization through the integration method of a customized adapter, the integration flexibility will be enhanced and will not be limited by back-end applications.
 
2. Database and file adaptor embedded :Through the indication of operation widget, the transaction system received will be rapidly converted to a corresponding database or file systematically to facilitate a swift development of back-end integration.
 
3. With AboveE’s Power Process BPM, the internal process integration system will be reinforced.By defaulting the process name developed onto the Power Link, the customers will be able to connect the partners’ data to internal handling process and to integrate with back-end system; Enabling a zero gap on data exchange between customers and their partners around the world.
 

Power Process BPM Suite Integration
 

Transaction Protocol Arrangement Mechanism

Agreement

Power Link provides graphic interface for administrators to determine data direction based on different PIPs and defines appropriate transaction mechanism according to the protocol agreed by you and partners; you will be able to proceed Level 1 and Level 2 testing after the protocol is set, the testing result will be beneficial to the development of back-end integration.
   

Simple Form Query & Monitoring Function

Message Query 
Through Power Link’s form query and monitoring function, the administrator or user will be able to use the receipt number to check transaction status directly, for example, the DocumentID shown on procurement message (3A4 Purchase Order Request) is the receipt number of purchase order. You will be able to resend the message through the query result if you would like to. Power Link also allows users to upload the definition file (xsl or xslt) to the system, which will be shown as per format set by the users when they proceed query on transaction contents.
   

Standard Message Generation Widget

Log Viewer

Power Link allows users to generate RosettaNet and AS2 compliant and other standard messages swiftly without a need to go through the programming process; while the standard message transmitted from partners will be obtained rapidly through the widget. Through the function, the adoption speed could be facilitated while manpower and time will be saved effectively.

Record Viewer

  

Power Link allows administrator to proceed query and viewing on the problematic message record and system event log, the administrator will be able to view the transaction message by DEBUG, INFO, WARN, ERROR and FATAL level to help discover the problem swiftly.
