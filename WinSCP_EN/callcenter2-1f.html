Developer Training Program (DTP)

Brief description of agenda:

Ten-day technical training is divided into a main course, two elective courses and WAD Workshop
   
Elective courses include basic Java and Java webpage development. The developers less familiar with Java application, through the elective courses, can learn how to write Java programs through the Eclipse development tool and can make use of simple JSP and JSTL to build a flexible flash-based webpage.

In the DTP (Developer Training Program), WAD (Web Application Development) is the kernel. WAD functions depending on J2EE and MVC (Model-View-Controller) in an industry-standard and open-source developer structure. WAD enables system module design, increases the benefit of development, lowers the maintenance cost, and enhances the reusability of a program. Through the training program, even if not knowing how to use a Power Process system and an API, developers may use the structure to easily create a form that connects to a Power Process BPM Suite system, significantly lowering the threshold of form development.

Course Aim: Fully participated in the program, participants will learn to generate a form used for the Power Process BPM Suite system through the WAD structure and use Java/JSP to develop, modify, and maintain the form, control logic, and data.
 

Objects:

Process designer and form designer having imported and being willing to develop Power Process BPM Suite

course hours:

10 days

Detailed description:

Basic Java Course

Brief description of agenda

 Java language is a programming language used for WAD Development Framework. This pre-course is designed especially for form developers not familiar with the Java language to use the WAD development form. In the basic Java, the syntax of Java programming language, data structure and the basic object-oriented concepts will be described.
In this course, participants will understand the basic object-oriented concepts and the Java programming syntax, and can use the Eclipse development tool to write a simple Java program.

Course hours:

Totally 2 days.
The course is elective for entry-level Java developer.
 
Java webpage development course

Brief description of agenda

 Java Web Development course will show the webpage structure, composed components and their operation modes, and bring you the knowledge and skill of flash-based webpage building through JSP and JSTL. After having completed the course, the participants will learn the basic syntax of HTML and Javascript and the operating mechanism of dynamic a webpage, and are able to write a JSP webpage.

Course hours:

Totally 1 days.
This course is elective for you to promote your ability in JSP programming.
 
WAD Training

Brief description of agenda

 WAD (Web Application Development) framework is a framework for programming a form. WAD framework will help form developers quickly build a sing form for Power Process BPM Suite system. WAD training course will show in detail the models and components that are used in this framework, and the relationship between the framework and the Power Process BPM Suite system. Through the implementation, students will learn how to simplify the cumbersome procedures and rapidly making all kinds of forms.
After completion of this course, participants will learn about the operation mode of WAD, and may develop, manage and adjust forms used for the system through this framework.

Course hours:

Totally 2 days.
This course is the kernel of “DTP developer training courses”.
 
WAD Workshop

Brief description of agenda

 After learning the WAD framework, customers may choose an upcoming production and an on-line process, and in this Workshop, senior advisers having experience in industries guide students to put WAD structure practice.

Course hours:

Totally 5 days.
This course is elective. After completion of this course, participants will have experience in practically operating the WAD framework building system and even complete building an on-line form system for enterprises.
 

Start Date:

depending on customers’ request
